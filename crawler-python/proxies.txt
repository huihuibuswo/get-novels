# =======================================================
# 代理列表配置文件 (proxies.txt)
# =======================================================
# 
# 在此文件中，每一行填写一个代理地址。
# 支持以下格式：
#   - IP:端口                     (例如: 112.112.112.112:8888)
#   - http://IP:端口              (例如: http://112.112.112.112:8888)
#   - http://用户名:密码@IP:端口    (例如: http://user:pass@112.112.112.112:8888)
#
# 程序启动或被目标站点风控重试时，会自动从该列表中“随机”选择一个代理使用。
# 每次修改此文件无需重启服务，下一请求会自动读取并生效。
#
# 请在下方添加您的代理列表（删除开头的 # 号即可启用）：
# -------------------------------------------------------
# 127.0.0.1:7890
# 112.112.112.112:8888
# 123.45.67.89:8080
# http://username:password@101.101.101.101:8888
